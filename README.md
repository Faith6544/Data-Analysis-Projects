# 📊 Data Analysis Projects — Atigah Faith

A collection of data analysis projects built with Python, SQL, and visualization tools — covering real-world business problems using Nigerian and African market datasets.

---

## 📁 Projects

| # | Project | Tools | Dataset Size |
|---|---|---|---|
| 01 | [Sales Performance Analysis](./01-sales-performance-analysis/) | Python, pandas, matplotlib, seaborn | 1,200 orders |
| 02 | [Customer Churn Analysis](./02-customer-churn-analysis/) | Python, SQL (SQLite), pandas, seaborn | 1,000 customers |
| 03 | [Nigeria E-Commerce Dashboard](./03-nigeria-ecommerce-dashboard/) | Python, pandas, matplotlib + Power BI exports | 1,500 orders |

---

## 🛠 Tools & Skills Demonstrated

- **Python** — pandas, NumPy, matplotlib, seaborn
- **SQL** — querying, filtering, aggregation
- **Data cleaning** — nulls, type conversion, feature engineering
- **EDA** — trends, distributions, comparisons
- **Visualization** — bar charts, line graphs, heatmaps, pie charts, KDE plots
- **Business Intelligence** — Power BI/Tableau-ready CSV exports
